### An E-commerce platform in the form of a Web Application incorporating concepts of DBMS along with PHP, HTML, CSS.

The ShopDaden Web Application was developed using HTML, CSS, JavaScript, PHP, phpMyAdmin, MySQL and XAMPP as a result of implementation of key concepts of DBMS such as normalization for an efficient database design.
